## [LG monitors silently install software via Windows Update](https://videocardz.com/newz/lg-monitors-silently-install-software-through-windows-update-without-user-consent) ⭐️ 9.0/10

LG monitors are silently installing software through Windows Update without user consent, as confirmed by Gamers Nexus using an LG UltraGear 34GX900A-B. This poses severe security and privacy risks because the installed software has full system access, runs at boot, and can be triggered simply by plugging in an LG monitor via HDMI. The software is installed automatically when a user plugs in a new or existing LG monitor, and it has internet access with no sandboxing, effectively acting as a potential backdoor.

hackernews · baranul · Jul 18, 10:21 · [Discussion](https://news.ycombinator.com/item?id=48956688)

**Background**: Windows Update can deliver driver and software packages from hardware vendors to ensure device compatibility. However, this mechanism can be abused to install unwanted or potentially malicious software without user awareness, similar to past autorun issues with USB drives.
