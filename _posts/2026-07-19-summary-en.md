## [LG monitors silently install software via Windows Update](https://videocardz.com/newz/lg-monitors-silently-install-software-through-windows-update-without-user-consent) ⭐️ 9.0/10

LG monitors are silently installing software packages through Windows Update without user consent, as confirmed by Gamers Nexus with an LG UltraGear 34GX900A-B. The software runs with full system access and starts at every boot. This poses a significant security and privacy risk, as it allows a third-party vendor to install software with full system access without user interaction. It could be exploited in a supply chain attack, affecting potentially millions of LG monitor users. The software installs automatically when an LG monitor is connected via HDMI, even if the monitor was previously connected. Workarounds include disabling automatic download of manufacturer apps via Group Policy or Device Installation Settings.

hackernews · baranul · Jul 18, 10:21 · [Discussion](https://news.ycombinator.com/item?id=48956688)

**Background**: Windows Update is designed to deliver drivers and software from hardware manufacturers to ensure device compatibility. However, this incident highlights a flaw where Microsoft does not sufficiently vet the software pushed by vendors, allowing potentially unwanted applications to be installed without user consent.
